# MFCenter

## Quick start

```
git clone https://github.com/Sagleft/MFCenter.git project
cd project
mkdir view/cache
chmod 777 view/cache
cp example.env .env
composer update
cd controller/public_html
cp example.htaccess .htaccess
cd ../../tools
sh get_depends.sh
```
